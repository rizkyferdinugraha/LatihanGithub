Selamat datang di Latihan GIT

Ini adalah contoh penggunaan GIT untuk version control.

Langkah-langkah:

  1. Inisialisasi repositori GIT dengan perintah git init.
  2. Tambahkan file ke staging area dengan perintah git add nama_file.
  3. Commit perubahan dengan perintah git commit -m "Pesan commit".
  4. Lihat status repositori dengan perintah git status.
  5. Lihat riwayat commit dengan perintah git log.
  6. Jika perlu, buat branch baru dengan perintah git branch nama_branch.
  7. Pindah ke branch tersebut dengan perintah git checkout nama_branch.
  8. Gabungkan branch dengan perintah git merge nama_branch.
  9. Jika ada konflik, selesaikan konflik tersebut dan lakukan commit lagi.
  10. Untuk mengirim perubahan ke repositori remote, gunakan perintah git push origin nama_branch.

Untuk informasi lebih lanjut, kunjungi dokumentasi resmi GIT.
